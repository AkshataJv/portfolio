# Personal Portfolio Website 🚀

A modern, responsive personal portfolio website showcasing my journey as a Software Engineer specializing in AI & Data Science. Built with clean HTML5, CSS3, and JavaScript to create an engaging user experience.

## 👋 About Me

Hi! I'm **Akshata**, a passionate Software Engineer from India with expertise in AI & Data Science. Currently pursuing my BTech in AI & Data Science at KKWIEER, Nashik (2024–2028). I love solving complex problems and building innovative solutions that make a difference.

## 🌟 Features

- **Responsive Design** - Fully optimized for desktop, tablet, and mobile devices
- **Interactive Navigation** - Smooth scrolling with mobile hamburger menu
- **Dynamic Tabs** - Interactive tabs for Skills, Education, and Projects sections
- **Portfolio Showcase** - Visual display of projects with hover effects
- **Contact Form** - Functional contact form integrated with Google Sheets
- **Social Media Integration** - Direct links to LinkedIn and GitHub profiles
- **Downloadable CV** - Easy access to resume download
- **Modern Animations** - Smooth transitions and hover effects throughout

## ⚙️ Contact Form Integration

The contact form is integrated with Google Apps Script for seamless message handling:

- Form submissions are automatically saved to Google Sheets
- Success/error messages provide user feedback
- Form validation ensures data quality
- Automatic form reset after successful submission

### Setting Up Contact Form

1. Create a Google Apps Script with the provided script URL
2. Update the `scriptURL` variable in the JavaScript section
3. Configure your Google Sheet to receive form data

## 🎨 Customization

### Updating Colors
The website uses CSS custom properties. Update the main color scheme in `style.css`:

```css
:root {
  --primary-color: #51bbe1;
  --text-color: #080808;
  --background-color: #fff;
}
```

### Adding New Projects
1. Add project images to the `assets/images/` folder
2. Update the portfolio section in `index.html`
3. Add corresponding styles in `style.css`

### Modifying Content
- Update personal information in the HTML file
- Replace profile picture and project images
- Modify the CV/resume file in the assets folder

## 🚀 Deployment

### GitHub Pages
1. Push your code to a GitHub repository
2. Go to repository Settings → Pages
3. Select source branch (main/master)
4. Your site will be available at `https://yAkshataJv.github.io/portfolio`

### Netlify
1. Connect your GitHub repository to Netlify
2. Set build command: None (static site)
3. Set publish directory: `/` (root)
4. Deploy automatically on every push
